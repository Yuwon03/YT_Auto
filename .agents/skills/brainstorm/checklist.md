# brainstorm Checklist

- Reference files (if used) are listed explicitly.
- `run_id` is explicit and inherited from upstream when available.
- Duration target (`short`, `medium`, `long`) is fixed.
- Concept attributes are explicit (audience, setting, conflict scale, POV, boundaries, series intent).
- Conflict, ending emotion, and twist timing are defined.
- At least 3 distinct directions are produced.
- Final chosen direction has continuity hook.
- Sensitive real-world topics are abstracted into fictional context.
- Markdown and json outputs are both written with lineage/handoff metadata.
