- Recommendation follows canonical order (`brainstorm -> idea_fetch_online -> story -> transcript -> tts`).
- Primary recommendation has a clear reason tied to missing artifact.
- Two alternative skills include explicit tradeoffs.
- Response includes a copy-pastable next command.
- Response includes explicit pipeline completion status.
