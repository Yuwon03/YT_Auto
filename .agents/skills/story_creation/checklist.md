# story_creation Checklist

- Exactly one brainstorm artifact is selected and readable.
- Exactly one online reference artifact is selected and readable.
- Upstream brainstorm artifact has `handoff_ready=true` (or override risk is documented).
- Upstream online reference artifact has `handoff_ready=true` (or override risk is documented).
- `run_id` is explicit and consistent with upstream.
- Runtime bracket (`short`, `medium`, `long`) is explicitly set.
- AI-user concretization notes are recorded before draft generation.
- Scene map has a clear purpose and state change for every scene.
- Reveal sequence is causal and consistent with earlier setup.
- Ending keeps at least one unresolved continuation hook.
- Story draft and scene map are both saved with lineage/handoff metadata.
