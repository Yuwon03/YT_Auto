# image_plan Checklist

- Source story file is explicit and readable.
- Source story artifact handoff status is checked.
- `run_id` is explicit and consistent with upstream.
- Chapter/image slots are complete for the planned runtime.
- Every slot has query keywords and at least two source candidates.
- License notes exist for every candidate.
- Low-confidence or missing-license slots are marked `needs_review`.
- Markdown and json plans are both generated with lineage/handoff metadata.
